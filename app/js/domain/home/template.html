Welcome to Synglossia!

This is an experimental language evolution modeling program.

Each language in the list below is the parent language of a linguistic structure here called a "syngloss." A syngloss is a structure that encapsulates all features of a parent language throughout their introduction into other languages, including descendant languages and languages in other families that borrow these features.

Click on any of these languages to view information about its syngloss.
